
# Git Safe Backup & Restore

## Backup (source of truth)
git add -A
git commit -m "msg"
git push origin main

## Restore (other device)
git fetch origin
git status
git log --oneline main..origin/main   # optional: see missing commits
git diff --stat origin/main           # optional: see file changes

# merge changes (keep local edits)
git pull origin main

# OR overwrite with remote (discard local edits)
git reset --hard origin/main
